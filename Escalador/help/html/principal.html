La pantalla inicial muestra una barra de menú que proporciona acceso a las diferentes funciones del programa. Estas funciones están agrupadas en dos bloques: Nuevo y Consulta.

El elemento de menú 'Nuevo' permite acceder a:

  • Configuración
  • Entrenamiento
  • Itinerario
  • Itinerario realizado

El elemento de menú 'Consulta' permite acceder a:

  • Configuracion
  • Entrenamientos
  • Itinerario
  • Itinerarios realizados
